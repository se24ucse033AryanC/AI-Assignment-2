## Objective

Design and develop a Simple Reflex Agent that reads environmental parameters from sensors and calculates the Air Quality Index (AQI).

---

## Agent Architecture

Sensors → Environmental data (PM2.5 value from file)

Agent Program → Calculates AQI category

Actuator → Displays AQI level

---

## How to Run

cd Group_Project/AQI_Agent  
python3 aqi_agent.py

## Example Output
